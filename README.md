# 🛡️ Ruqyah Links (Minimalist Directory)

কোরআন ও সুন্নাহ ভিত্তিক শুদ্ধ আমল এবং শক্তিশালী রুকইয়াহর একটি অত্যন্ত সাদামাটা, পরিষ্কার এবং মিনিমালিস্ট ভিডিও ডিরেক্টরি।

## 🚀 Live Demo
[https://rabiul29.github.io/ruqyah-links](https://rabiul29.github.io/ruqyah-links)

## 💎 কেন এটি মিনিমালিস্ট?
- **Less is More:** কোনো অতিরিক্ত রং, গ্রাডিয়েন্ট বা শ্যাডো নেই। শুধুমাত্র সাদা ব্যাকগ্রাউন্ড এবং কালো টেক্সট।
- **Text-Focused:** পুরো মনোযোগ শুধুমাত্র ভিডিওর শিরোনাম এবং ইউটিউব লিংকের ওপর।
- **Zero Distraction:** অযথা কোনো ব্যানার বা সাইডবার নেই।
- **Blazing Fast:** অত্যন্ত হালকা কোড, যা দ্রুত লোড হয়।

## 📂 ফাইল স্ট্রাকচার
- `index.html`: মূল মিনিমালিস্ট কাঠামো।
- `style.css`: সাদামাটা ডিজাইন ও রেসপন্সিভনেস।
- `script.js`: ডাইনামিক ডেটা হ্যান্ডলিং।
- `links.json`: আপনার সকল ভিডিও ডেটা।
